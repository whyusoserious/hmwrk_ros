# Как использовать?

Клонировать
```
git clone https://github.com/whyusoserious/hmwrk_ros.git
```
Перейти в клонированную директорию
```
cd hmwrk_ros
```
Настроить среду выполнения
```
rm -rf build
catkin_make
source devel/setup.bash
```
Запустить проект
```
roslaunch turtle launch.launch
```
ver: melodic
